📐 Notas de diseño de este esquema

  • Las cards de conductores son clicables — al hacer clic se actualiza el panel de detalle inferior
  • El borde de color de cada card indica el umbral: verde ≥80%, amarillo ≥50%, rojo <50%
  • El sparkline dentro de cada card muestra la tendencia de las últimas semanas (sube/baja)
  • La tabla izquierda ordena peores primero para facilitar la acción correctiva
  • La tabla derecha muestra el desglose de envíos retrasados del conductor seleccionado, con el tracking ID, seller, fechas y comuna — para investigar causas
  • Cuando llegue el Excel con fecha_retiro real, el cálculo se actualizará automáticamente
  • Se puede agregar un filtro por zona/comuna para detectar si los retrasos son geográficos
